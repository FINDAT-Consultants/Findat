# Course media

Place the original course recording in this folder using this exact filename:

- `Data.MP4`

A placeholder `Data-Thumbnail.jpg` is included so the page has a valid poster image. Replace it with your preferred thumbnail while keeping the filename unchanged.

Large MP4 files may exceed GitHub's normal file-size limit. For large recordings, use Git LFS, GitHub Releases, or external video hosting and update the paths in `index.html`.
